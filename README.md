# Keenkeeper is a kind of social media application.

It is an interesting project for me. All data is shown on screen through the fetching API in this project. With this application, you can call, video call, and text your friend. On the initial page, you can see your friend list. Click on a friend's profile, and the friend's detail page will show where you can see all details about your friend. In the detail page, there are three buttons call, text, and video. If you click on them, some interaction will happen, like calling, video, or texting. If you want to see interaction with details, you have to jump on the timeline page, where you can see call video and text history in real time. How much interaction happened, you can see on the stats page with an amazing pie chart. You can easily understand the situation of your interaction with your friends.

# Some features of this project are:

- Sort data by video, call, and text in the Timeline page. Initially, all category data are shown together.
- can see when you communicate with your friend, in the timeline page.
- You can observe the situation about communication at a glance with a pie chart
- For a better user experience, empty massage show.

The technology used to create this project is React.js, React Router, Tailwind CSS, and DaisyUI, etc.
